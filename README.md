MODIS
===

Aplicación web de Shiny para visualizar figuras MODIS del Río de la Plata.

Requisitos
===

library(shiny)
library(RCurl)
library(jpeg)
library(raster)
library(maptools)
library(rgeos)
library(rgdal)
